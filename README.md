🧑‍⚕️ Prescripto – Book Appointment With Trusted Doctors

An advanced Doctor Appointment Booking System built using modern web technologies. This platform allows patients to book appointments with verified doctors, manage schedules, and access healthcare services easily.

🚀 Features
👨‍⚕️ User Registration & Login (Patient/Doctor/Admin)
📅 Book & Manage Appointments
🩺 Doctor Listing with Specialization
📊 Admin Dashboard for Management
🔐 Secure Authentication System
📱 Responsive UI (Mobile + Desktop)
🗂️ Organized appointment tracking system


🛠️ Tech Stack
Frontend: React.js / HTML / CSS / JavaScript
Backend: Node.js / Express.js
Database: MongoDB
Authentication: JWT / Cookies
Styling: Tailwind CSS / CSS3

⚙️ Installation & Setup
1️⃣ Clone the repository
git clone https://github.com/VishnuKumarSingh/Prescripto__Book-Appointment-With-Trusted-Doctors.git

2️⃣ Go to project folder
cd Prescripto__Book-Appointment-With-Trusted-Doctors

3️⃣ Install dependencies
Backend
cd backend
npm install
Frontend
cd frontend
npm install

▶️ Run the Project
Backend
npm start
Frontend
npm run dev
